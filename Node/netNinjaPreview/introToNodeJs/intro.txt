What Is Node.Js?

- A platform which allows us to run Javscript on a computer/server
- Read,delete, and update files
- Easily communicate with a database

Why would we want to use it?
- it uses javascript
- very fast (runs on the v* engine and uses on-blocking code)
- Huge ecosystem of open source packages (npm)
- Great for real-time services (like chats)
